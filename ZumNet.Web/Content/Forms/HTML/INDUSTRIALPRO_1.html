산업재산권 유지 평가의 목적
 본 평가표는 발명의 필요성을 재평가하여 불필요한 권리유지로 발생하는 비용을 억제하고,
현재까지도 유효한 고안인 경우 그 활용을 재고(再考)하고자 합니다.
만일 유지하지 않더라도, 본 발명에 대한 독점권만 소멸되는 것으로, 본 발명을 사용할 수 없는 것은 아닙니다.

검토 대상 권리
   ● 구분 : 특허, 실용신안, 디자인
   ● 명칭 :
   ● 출원일 / 출원번호 :
   ● 등록일 / 등록번호 :
   ● 권리 만료 예정일 :

검토 내용
   ● 기술단계 : 제조/판매, 상품화 완료, 시제품 보유, 사업화 준비, 기타( )
   ● 사용관계 : 사용중, 사용계획중, 타사 사용허락중, 사용하지 않음, 사용 의사 없음, 기타( )
   ● 진보성 : 획기적인 기술임, 효과가 독특함, 회피가 용이하지않음, 진부화하였음, 기타( )
   ● 대외성 : 대외적 영향 있음, 대외적 영향 없음,

검토 의견:                                      

검토 결과 : 독점권 유지, 독점권 소멸
